# 陽交光復學餐選擇機

本程式為陽明交大 JAVA 期末專案，由個人獨立設計，並開發完成。

此為一款專為**陽明交大光復校區學生**與**鄰近竹科上班族**設計的校內餐廳選擇輔助程式，以 Java 開發，整合校內三間餐廳的完整店家資訊，透過多重篩選與視覺化隨機抽選功能，解決高峰時段選餐耗時、資訊查詢不便等問題。

---

## 開發背景

校內餐廳資訊分散且不齊全：Google Maps 缺少許多餐廳內的店家細節，學校官網查詢流程繁瑣費時。這導致了幾個常見問題：

- 新生不熟悉各餐廳供餐時段與店家選項，需要實地查看才能了解
- 舊生長期面臨選擇疲勞，最終傾向選擇外送服務或超商便利店，增加就餐成本
- 竹科上班族到校內用餐時，缺乏快速掌握餐廳資訊的工具

本專案的目標是將三間校內餐廳的資訊整合至單一介面，讓使用者不需要實地查看或跨平台查詢，即可快速做出用餐決策。

---

## 功能說明

**多重篩選條件**

使用者可依照「用餐時段」與「餐點類型」進行篩選，快速縮小符合當下需求的店家範圍。篩選結果即時更新，不需重新啟動程式。

**視覺化隨機抽選**

針對選擇困難的使用者，程式內建隨機抽選功能，以動畫方式呈現抽選過程，抽出結果後自動顯示於「用餐時段店家資訊」欄位，增添趣味性的同時也輔助最終決策。

**即時時間顯示**

介面右下角持續顯示當下時間，讓使用者能直接對照各餐廳的供餐時段，不需額外查詢即可判斷目前可選擇的店家。

**整合三間校內餐廳資訊**

- 二號學生餐廳
- 女二舍餐廳
- 研三舍餐廳

---

## 目標使用者

**校內新生**

新生對校內餐廳環境不熟悉，不知道各餐廳在不同時段提供哪些店家。本程式透過時段篩選與即時時間顯示，讓新生能直觀了解當下的供餐選項，減少實地查看所浪費的時間，也有助於降低非就餐時段學生湧入餐廳造成的人潮問題。

**校內舊生**

長期在固定幾個選項之間循環容易產生選擇疲勞，進而轉向外送或超商。本程式的隨機抽選功能提供新鮮感，幫助舊生重新發現校內餐廳選項，降低對外送的依賴與就餐成本。

**竹科上班族**

光復校區鄰近竹科，吸引不少上班族到校內用餐。本程式整合完整餐廳資訊並支援時段篩選，讓上班族能在有限的午休時間內迅速掌握可用的餐廳選項，提升用餐效率。

---

## 技術說明

- **開發語言**：Java
- **介面**：GUI 視覺化介面，含隨機抽選動畫與即時時間顯示
- **資料**：靜態整合三間校內餐廳店家資訊（含供餐時段與餐點類型分類）
- **防呆機制**：非法輸入提示，避免程式異常中斷

## 使用方式

1. Clone 此專案

   ```bash
   git clone https://github.com/YOUR_USERNAME/nycu-cafeteria-selector.git
   ```

2. 以 IDE（如 IntelliJ IDEA 或 Eclipse）開啟專案

3. 執行主程式

4. 依需求選擇用餐時段與餐點類型，或直接點選隨機抽選功能
